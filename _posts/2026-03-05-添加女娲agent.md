# 添加女娲 Agent

今天配置了一个新的 AI 助手，命名为"女娲"。

## 背景

之前使用的助手名为"小虾米"，今天决定换一个更有意义的名字。

## 配置过程

### 1. 修改身份信息

编辑 `/root/.openclaw/workspace/IDENTITY.md`：

```markdown
- Name: 女娲
- Emoji: 🧚
```

### 2. 配置用户偏好

在 `USER.md` 中记录了交互偏好：

- 所有记录使用中文
- 修改文件时展示修改过程
- 系统文件操作先询问

### 3. 项目管理工作流程

建立了标准化的项目创建流程：

1. 讨论想法 → 确定项目名称和范围
2. 创建项目目录（`~/workspace/project-name/`）
3. 创建飞书群组（使用 API）
4. 在群组中创建文档和表格

## 飞书群组自动化

通过 API 自动创建飞书群组的调研：

- 使用 `tenant_access_token` 认证
- 调用 `POST /open-apis/im/v1/chats`
- 权限：`im:chat:create`
- 测试通过：成功创建测试群组

## 心得体会

1. **文档化很重要**：将流程记录在 `TOOLS.md` 中，下次可以直接参考
2. **权限管理**：飞书应用的权限需要在开放平台申请
3. **自动化优先**：能自动化的就不要手动操作
